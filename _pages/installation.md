`aomaker` 已上传到`pypi` ，所以安装非常简单。

- 第一次安装

```bash
pip install aomaker
```

- 安装最新版本

```python
pip install -U aomaker
```

- 查看是否安装成功

直接命令行输入`aomaker` 

```bash
aomaker
```

能看到`aomaker` logo即安装成功。
